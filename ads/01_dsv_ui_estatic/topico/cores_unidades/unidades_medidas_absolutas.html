Voltar
 

As unidades de medida absolutas não possuem dependência em relação a outras unidades, elas são fixas e são apresentadas da forma como são determinadas, não sendo necessários valores de referência (W3SCHOLLS, ONLINE).

As unidades de medida que abordaremos são: cm (centímetros), mm (milímetros), in (polegadas, do inglês "inches"), px (pixels), pt (pontos), pc (picas).


Utilizando a unidade absoluta "cm"

Essa unidade do sistema métrico é usual em nosso dia a dia. Podemos determinar os comprimentos e tamanhos dos elementos que desejamos formatar utilizando essa unidade de medida.

Utilizando a unidade absoluta "mm"

Dez milímetros equivalem a um centímetro, as unidades são múltiplas entre si. O uso é análogo ao exemplificado anteriormente.

Este cabeçalho tem uma fonte de 0.75mm

Utilizando a unidade absoluta "in"

A polegada, ou inch, é uma unidade de medida que equivale a 2,54 cm ou 96 pixels.

Utilizando a unidade absoluta "px"

O pixel (px) corresponde à menor unidade que forma uma imagem na sua tela, por exemplo. Essa unidade é formada por três elementos de cor vermelho, azul ou verde; dependendo da luminescência de cada um deles, é apresentada uma cor diferente para aquele ponto. (TECMUNDO,ONLINE).

Um pixel equivale a (1/96) polegada.

Os pontos, ou pt, é uma unidade de medida que equivale a 1/72 de uma polegada.

As picas, ou pc, é uma unidade de medida onde 1pica (pc) equivale a 12 pontos (pt).
